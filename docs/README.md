## 名字的由来
- `蜜獾`(huan),绰号 `平头哥` 被称为最无所畏惧的动物;
- 它的个性：哥的一生不是在打架，就是在去打架的路上。管你是谁，生死看淡不服就干;
- 当下社会过于浮躁和虚假，我们敬畏这种(人/动物),虽说[自古深情留不住，唯有套路得人心],但我们还是期望人与人之间，少一些欺骗/套路，多一点真诚，任何事都坦然面对，拥抱变化、不惧未来。
